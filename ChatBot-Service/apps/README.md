# 聊天机器人服务（Chat Bot Service）

基于 **FastAPI**开发的简单聊天机器人服务。

## ✨ 功能特性

* 🚀 **流式对话**：逐字输出，打字机效果，体验流畅
* 🎨 **现代 UI**：渐变紫色主题，响应式设计，支持移动端
* 🧠 **通义千问**：基于阿里云百炼平台，支持 qwen-max/plus/turbo 多模型切换
* 📦 **轻量级**：FastAPI + 原生 JavaScript，无重型框架依赖

## 🛠️ 技术栈

* **FastAPI** - 高性能 Web 框架
* **Pydantic** - 请求参数校验 & 数据格式化
* **Uvicorn** - ASGI 生产级服务器
* **Python-dotenv** - 环境变量管理

## 🚀 快速开始

### 环境要求

* Python 3.13+
