# 博客系统后端（Blog System Backend）

基于 **FastAPI + Tortoise-ORM + MySQL + JWT** 的纯后端博客系统，提供完整的用户、文章、评论、点赞业务能力与接口服务。

## ✨ 功能特性

* 👤  **用户管理** （注册、登录、信息修改、软删除、个人资料）
* 🔐  **JWT 认证** （双 Token：Access Token + Refresh Token，自动过期 / 刷新）
* 📝  **文章管理** （发布、修改、删除、列表查询、详情查询、权限控制）
* 💬  **评论管理** （发布评论、删除评论、关联文章）
* 👍  **点赞管理** （文章点赞 / 取消点赞、点赞统计）
* 🛡️  **权限控制** （仅作者可修改 / 删除自己的内容）
* 🎯  **统一响应** （全局接口格式统一、异常统一封装）
* 🗑️  **软删除支持** （数据不物理删除，安全可恢复）

## 🛠️ 技术栈

* **FastAPI** - 高性能 Web 框架
* **Tortoise-ORM** - 异步数据库 ORM
* **MySQL** - 业务数据存储
* **JWT** - 接口鉴权与身份认证
* **Pydantic** - 请求参数校验 & 数据格式化
* **Uvicorn** - ASGI 生产级服务器
* **Python-dotenv** - 环境变量管理

## 🚀 快速开始

### 环境要求

* Python 3.13+
* MySQL 5.7+
* Redis 3.0+
